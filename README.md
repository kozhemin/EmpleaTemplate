## How to build

1) Clone the repository

```bash
$ git clone git@github.com:kozhemin/EmpleaTemplate.git
```

2) Install pods

```bash
$ cd EmpleaTemplate
$ pod install
```

3) Open the workspace in Xcode

```bash
$ open "Emplea.xcworkspace"
```

4) Compile and run the app in your simulator
